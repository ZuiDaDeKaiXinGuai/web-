#### Undefined类型

> Undefined只有一个值，即特殊的undefined
```
var message;
alert(message == undefined);    //true
var message = undeﬁned;

```
> 包含undefined值的变量与尚未定义的变量不同

```
var message;                            
alert(message);  //"undefined"

```
> 对未初始化和未声明的变量执行typeof操作符都返回undefined值


