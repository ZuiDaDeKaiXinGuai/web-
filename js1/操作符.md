#### 一元操作符

> 只能操作一个值的叫一元操作符

- [x] 前置直接返回运算过后的值


- 递减


```
var age=20;

--age  等价于  age = age - 1

//age=19

age+1

//20

```



- 递增


```
var age=20;

++age  等价于  age = age + 1

//age=21

age+1

//22

```
- [x] 后置先返回原值在返回运算过后的值

- 递减


```
var age=20;

age-- 

//age=20

age+1

//20

```



- 递增


```
var age=20;

age++

//age=20

age+1

//22

```

#### 一元加和减操作符

> 一元加操作符以一个+号表式，放在数值的前面，对数值不会产生任何影响


> 一元减操作符用于表示负数



>一元加和减操作符主要用于基本的算术运算，用于数据类型转换

#### 布尔操作符

##### 逻辑非

```
!A

无论什么值什么类型都会返回布尔值

alert(!false); // true 
alert(!"blue"); // false 
alert(!0); // true 
alert(!NaN); // true 
alert(!""); // true 
alert(!12345); // false


alert(!!"blue"); //true 
alert(!!0); //false 
alert(!!NaN); //false 
alert(!!""); //false 
alert(!!12345); //true 

```
