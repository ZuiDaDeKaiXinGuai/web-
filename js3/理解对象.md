- 删除属性

```
delete 对象.属性名;
```






- 检测属性


```
属性名 in 对象

对象.hasOwnProperty(属性名)
```

- 枚举属性


```
Object.defineProperty(对象,'属性名',{
        //可编写
        writable:false/true,
        //可枚举
	enumerable:false/true,
	//值
	value:'属性值',
	//可配置
	configurable:false/true
})
```

- 序列化

```
//对象转换字符串；
JSON.stringify(对象);
//字符串转对象
JSON.parse(字符串);
//与地区对应
对象.toLocaleString();
```
